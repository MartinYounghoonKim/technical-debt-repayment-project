## Collections Framework

### Collections Framework
- 일종의 컨테이너라고도 표현함
- Iterator 는 Collections 인터페이스 안에 정의 되어져 있음. 그렇기 때문에 모든 Collections Framework는 이 메소드를 구현하고 있음을 보장한다.

<img src="./static/2160.png" />

#### Set
- 중복을 허용하지 않는 데이터 구조

#### List
- 개발자가 추가하려는 모든 값들을 Add 할 수 있음

#### Map
- 키벨류(Key-Value) 형태의 데이터 구조
